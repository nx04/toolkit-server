https://nodejs.org/en/download

## 简介

https://nodejs.org/dist/v24.11.0/node-v24.11.0-linux-x64.tar.xz

```
tar -C /usr/local -xvf  node-v24.11.0-linux-x64.tar.xz

## 设置环境变量
echo 'export PATH=$PATH:/usr/local/node/bin' >> /etc/profile


## 更新
source /etc/profile


## 检测是否安装成功
node version

```
